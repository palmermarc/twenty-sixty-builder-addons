== Description ==

2060 Digital Page Builder adds in custom functionality into the custom page builder for our clients.

Custom functionality includes, but is not limited to the following:

* Frequently Asked Questions
* Locations
* Staff Members

== Installation ==

1. Ensure that the 2060 Digital Page Builder is activated in the 'Plugins' menu in WordPress.
2. Upload the unpacked ZIP file that contains this plugin, or install the plugin from our GitHub repository.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Turn on the features that you would like to use on your site.

== Frequently Asked Questions ==

= How can I configure what items are displayed in the Page Builder? =

To configure the modules that your Page Builder uses, hover over Settings, and then click `2060 Digital Page Builder`. From there, click on Modules, and then check/uncheck the items you'd like to change.
